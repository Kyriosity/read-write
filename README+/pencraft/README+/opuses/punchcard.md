## Intramural experience - short and vintage

It was long after the punch decline and the last year they were legacy coursework at my university. I guess because the supply of this carton product became extinct.

### The task

It was an exercise in PL/I&nbsp;<sup>ğŸ”¢</sup> - output a curve of an assigned function (some $`ax^2 + bx + c`$ was mine). And the _output_ meant **direct to paper** using teletype symbols for X/Y axes and the graph. Believe it or not, there was no computer monitor in this story.
